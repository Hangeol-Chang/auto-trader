🤖 Discord Bot API

디스코드 봇과의 통합을 위한 API 엔드포인트

  • GET /api/discord/health
  • GET /api/discord/commands
  • POST /api/discord/webhook
  • POST /api/discord/notification

📈 TradingView API

TradingView 웹훅을 처리하는 신규 API 엔드포인트

  • GET /api/tradingview/health
  • POST /api/tradingview/signal (신규)
  • POST /api/tradingview/signal-test
  • GET /api/tradingview/strategies
  • GET /api/tradingview/webhook-config
  • GET /api/tradingview/signals/history

💰 Trading API

암호화폐 및 주식 거래를 제공하는 API 엔드포인트

  • GET /api/trading/health
  • GET /api/trading/balance
  • GET /api/trading/markets
  • POST /api/trading/orders
  • GET /api/trading/positions
  • GET /api/trading/performance

📈 Stock Trading API

KIS API를 통한 한국 주식 거래 엔드포인트

  • POST /api/trading/stock-signal
  • GET /api/trading/stock-balance
  • GET /api/trading/stock-price/{code}
